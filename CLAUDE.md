# Frontend Design Skill

Read and follow the design guidelines in .agents/skills/frontend-design/SKILL.md before starting.

## Key Principles
- NEVER use generic AI aesthetics (purple gradients on dark bg, Inter font, cookie-cutter layouts)
- Choose BOLD, distinctive typography (not Inter, Roboto, Arial)
- Commit to a clear aesthetic: Apple-like refined minimalism
- Motion should be scroll-linked and bidirectional (not just fade-in on entry)
- Use CSS variables for consistency
- Production-grade, accessible code

## Design References
- Apple MacBook Neo (apple.com/br/macbook-neo/): Clean, minimal, large typography, pinned scroll sections, video scrubbing
- AppyCamper (appycamper.com): Scroll-triggered animations that play FORWARD and REVERSE as you scroll up/down, pinned sections, Lenis smooth scroll
