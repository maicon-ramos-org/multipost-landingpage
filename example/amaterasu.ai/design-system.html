08. Implementation Notes

Fluid Scaling

This system uses vw (Viewport Width) units for everything. To maintain the exact look of the original design, do not convert these to px or rem without careful planning. The design logic assumes a base of 1440px for desktop and 375px for mobile.

Scoped Styles (Vue/Nuxt)

Original components use scoped CSS attributes like data-v-a8a1a084. When porting these components to new environments, you can either keep these attributes or rename them to standard class names while ensuring the CSS selectors in assets/index_cc3f93b4099a.css are updated accordingly.

Motion Curves

Brand animations exclusively use --ease-in-out-quart for surface transitions and linear for continuous patterns (like the waves and blinkers).
